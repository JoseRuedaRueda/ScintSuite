32  Adapted from: https://www.aug.ipp.mpg.de/aug/manuals/map_equ/

34  @param shot: Shot number
35  @param Rin: Array of R positions where to evaluate (in pairs with zin) [m]
36  @param zin: Array of z positions where to evaluate (in pairs with Rin) [m]

39  @param ed: edition, default 0 (last)
40  @param tiniEQU: Initial time to load the equilibrium object, only valid

42  @param tendEQU: End time to load the equilibrium object, only valid if
43  equ is not pass as input (in s)
44  @param time: Array of times where we want to calculate the field (the
45  field would be calculated in a time as close as possible to this
46  @param equ: equilibrium object of clas EQU (see
47  https://www.aug.ipp.mpg.de/aug/manuals/map_equ/equ/html/classsf2equ__20200525_1_1EQU.html)
48  @return br: Radial magnetic field (nt, nrz_in), [T]
49  @return bz: z magnetic field (nt, nrz_in), [T]
50  @return bt: toroidal magnetic field (nt, nrz_in), [T]
51  @return bp: poloidal magnetic field (nt, nrz_in), [T]
52  """
53  # If the equilibrium object is not an input, let create it
54  if equ is None:
55  equ = EQU(shot, diag=diag, exp=exp, ed = ed, tbeg=tiniEQU,
56  tend=tendEQU)
57  # If the equilibrium object is an input, check if the shot number is correct
58  else:
59  if equ.shot != shot:
60  print('Shot number of the received equilibrium does not match!')
61  br = 0
62  bz = 0
63  bt = 0
64  bp = 0
65  return br, bz, bt, bp
66  # Now calculate the field
67  br, bz, bt = meq.rz2brzt(equ, r_in=Rin, z_in=zin, t_in=time)
68  bp = np.hypot(br, bz)
