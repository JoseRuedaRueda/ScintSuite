Detailed Description

This is a sample scrip which load a reference frame, load a video, calculate and aling the strike maps and remap the scintillator signal.

Created as an example to use the routines without the graphical user interface

DISCLAIMER: This was created on the 21/10/2020. Since them several improvement may have been done, it is possible that some function has been changed and the script does not work at all now. If this happens, contact jose rueda (jose..nosp@m.rued.nosp@m.a@ipp.nosp@m..mpg.nosp@m..de) by email
