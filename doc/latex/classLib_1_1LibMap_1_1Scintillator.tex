\doxysubsection{Detailed Description}
Class with the scintillator information. 

see Scintilator\+\_\+plate.\+py for extra documentation on the format

Note, the notation is given by FILDSIM, and it is a bit misleading, in FILDSIM x,y,z axis are defined, the scintillator lies in a plane of constant x, so the only variables to play with are y,z. However, x, y are always used to refer to x horizontal and vertical direction in the camera sensor. We have to live with this. Just ignore the x coordinates of the scintillator data and work with y,z as they were x,y
