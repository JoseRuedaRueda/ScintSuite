Detailed Description

Load a frame, load the calibration and remap the scintillator signal.

Created as an example to use the routines without the graphical user interface

DISCLAIMER: This was created on the 21/11/2020. Since them several improvements may have been done, it is possible that some function has been changed and the script does not work at all now. If this happens, contact jose rueda (jose..nosp@m.rued.nosp@m.a@ipp.nosp@m..mpg.nosp@m..de) by email

DISCLAIMER 2: The script is quite slow because of the AUG routine which reads the magnetic field... is not fault of the suite!
