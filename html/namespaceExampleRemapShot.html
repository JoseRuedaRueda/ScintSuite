Detailed Description

Load a video and perform the remaping for each frame.

Created as an example to use the routines without the graphical user interface

DISCLAIMER: This was created on the 11/11/2020. Since them several improvements may have been done, it is possible that some function has been changed and the script does not work at all now. If this happens, contact jose rueda (jose..nosp@m.rued.nosp@m.a@ipp.nosp@m..mpg.nosp@m..de) by email
